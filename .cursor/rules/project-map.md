# Project Map – Berntsen Struktur

## Project overview
This repository contains the website for **berntsenstruktur.no**.

The purpose of the site is to help people structure ideas, projects and businesses.
It acts as a consulting landing page.

The design philosophy:
- minimal
- fast loading
- clear messaging
- strong call-to-action

---

# Tech stack

Frontend:
- HTML
- CSS
- minimal JavaScript

Hosting:
- GitHub Pages

Domain:
berntsenstruktur.no

index.html

# Landing page structure

The landing page should contain these sections:

1. Hero  
2. Services  
3. Process  
4. Who this is for  
5. Contact / call to action

Cursor should maintain this structure when modifying the page.

---

# Coding rules

When editing this repository:

- Do not break existing functionality
- Prefer simple HTML over frameworks
- Keep CSS readable and structured
- Avoid unnecessary dependencies
- Do not add build systems unless requested
- Optimize for clarity and speed
